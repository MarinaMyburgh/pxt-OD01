# show string on OD01

Displays a string on the OD01 module.

```sig
OD01.showString("hello, micro:bit!")
```

The ``init`` block must be placed before this.

## Parameters

* **text**: the string to display.

## Example

```blocks
OD01.init()
OD01.showString("hello, micro:bit!")
```
